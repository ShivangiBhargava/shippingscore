# Ship Score

Ship Score is a brutally honest AI critic for landing pages. You paste a URL and it returns a 0–100 score across four axes — Product Thinking, Craft, Originality, and Shippedness — plus a one-paragraph roast and a verdict: Ship It, Fix It First, or Kill It. It's built for indie hackers, hackathon entrants, and founders who are tired of polite feedback from friends and want the kind of read a senior PM at Linear or Stripe would give them. Built with Lovable (TanStack Start + React), Google Gemini 3 Flash via the Lovable AI Gateway, and Novus.ai for behavioral analytics. The biggest lesson shipping it: getting an LLM to be specific and mean-but-fair is 90% prompt engineering and 10% model choice — vague graders are worse than no graders.

Ship Score is live, and every critique is tracked. Novus.ai is installed at the root level, logging every session, URL submission, and score. Here's the dashboard proving real user analytics—turning 'we track' into 'here's the data.
